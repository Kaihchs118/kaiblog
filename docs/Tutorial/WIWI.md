剛從[社群媒體](/blog/why-i-dont-use-social-media)那個「所有文字都得是預設格式」的排版地獄爬出來嗎？

歡迎來到獨立部落格的自由世界！在[部落格文體](/blog/social-vs-blog-content)中，[文字格式](/blog/freedom-to-format)不只是裝飾，它是你語氣的延伸、對讀者[情緒勒索](/blog/pua)的工具，也是修辭不可或缺的重要部份。

突然之間有了一整個 HTML / CSS / [Markdown](/blog/why-use-markdown) 工具箱的你，可能需要一點時間適應這樣的自由。就讓這篇「部落格文體的格式修辭學」指南來幫助你吧！

---

### **粗體** 

想要讓讀者感受到你內心的吶喊時使用。

範例：我今天去 7-11 買咖啡，店員問我要不要買衛生紙，我說不用，然後他**又問了三次！**

### *斜體*

許多人說中文不該用斜體，但誰理他們啊！這種歪歪斜斜、難以閱讀的中文，最適合用在圖片說明、文章中註解（注意不要與「註腳」混淆）、謎之音或悄悄話等處。

範例：*（非當事衛生紙，這一串是送的）*

### ~~刪除線~~

把真心話寫出來，然後收回去，假裝對方看不到，但其實就是想要對方看到。

範例：最後我還是買了兩串衛生紙，因為~~店員很正~~我是個心地善良的人。

### 註腳

就是那些文章中會出現的小數字。正文寫不下的碎碎念最佳去處，還可以順便展現學術氣息！

範例：反正家裡多囤一點衛生紙又不會怎麼樣。[^1]

### [超連結](/blog/hyperlinks)

部落格的靈魂所在！可以連到自己以前的文章、別人的文章，或者 Rick Astley 的 [*Never Gonna Give You Up*](https://www.youtube.com/watch?v=dQw4w9WgXcQ)。

範例：衛生紙用完跟[停水](/blog/water-outage)，到底哪一個比較可怕呢？

### 引言區塊

把廢話放到這裡面，權威感瞬間 up up！

範例：

> 「人生就像一捲衛生紙，看起來很多，但拉個肚子就用完了。」
>
> — 我說的

### 符號清單

當你意識到這段文字太長，讀者可能會睡著時，把它拆成清單。這樣讓你看起來很有條理，雖然內容完全不是那麼一回事。

範例：

我買衛生紙的時機：

- 想到就買
- 有特價就買
- 還拿得動就買

### 數字清單

跟符號清單很像，但多了順序性功能。

範例：

如何買衛生紙：

1. 走進 7-11
2. 發現衛生紙其實放在店門口外面
3. 走出 7-11
4. 拿六串衛生紙
5. 再度走進 7-11
6. 結帳，店員說其實你沒必要把六串都拿進來，跟他說就好
7. 拿著六串衛生紙走出去

### 表格

只要加上框線，任何主觀偏見看起來都像是客觀分析。

範例：

| 品牌 | 價格 | 評價 |
| :--- | :--- | :--- |
| 舒X | 貴 | 很厚，失戀大哭時必買 |
| 春X | 便宜 | 記得第一擦要用三張，不然會破 |
| Unixxxx | 中等 | 在 7-11 買很多東西時會送 |

### 小字註解

原本是用來標註讀音的，但現在是「口是心非」時的專用格式。

範例：快給我一包衛生紙，因為我<ruby>要把桌子擦乾淨<rt>昨天吃了麻辣鍋</rt></ruby>。

### 標題階層

用來組織文章架構，或者假裝你的文章很有架構。

範例：

# 衛生紙囤貨指南
## 為什麼要囤？
### 因為會漲價
#### 雖然也不一定會漲
##### 但囤了就是很開心

### 螢光筆

因為你覺得粗體還不夠。

範例：衛生紙是<mark>**最值得囤的物品**</mark>，不會過期，怎麼囤都不會浪費！

### 行內程式碼

用在想用「等寬字體」呈現，或是純粹想要文字有個外框時。我都拿來當作「音名」的標記來用。

範例：G7 和弦的組成音是 `G`、`B`、`D` 和 `F`，雖然這跟衛生紙一點關係都沒有。

### 程式碼區塊

讓你看起來像是軟體工程師，或者也可以用來展示任何你想要用「等寬字體」呈現的內容。

範例：

```javascript
function 該買衛生紙嗎() {
  if (家裡剩餘 < 10捲) {
    return true;
  }
  if (特價中) {
    return true;  
  }
  return true;
}
```

### 鍵盤標示

教讀者如何操作電腦，或者用來炫耀你記得快速鍵。

範例：為了搶到特價衛生紙，他不斷狂 <kbd>Ctrl</kbd> + <kbd>R</kbd> 刷新頁面。

### 模糊文字

防雷神器！或者用來遮住壞壞的文字。

範例：那天我在超市遇到 Wiwi，他的推車裡有<span className="blur-text">十串衛生紙</span>，真是個奇怪的男人。

### 折疊區塊

因為越把東西藏起來，讀者就會越想點開來看。

範例：

<details>

<summary>讓你驚訝的衛生紙冷知識！</summary>

根據[這個衛生紙製造商的網站](https://uk.nakedpaper.com/blogs/news/does-70-of-the-world-not-use-toilet-paper)，其實世界上有 70% 的人上大號是不用衛生紙的，他們優先用水來清潔。在亞洲許多國家、中東、南美洲和南歐，免治馬桶、手持沖洗器、甚至只是一壺水，才是廁所的標準配備。

其實我也是那 70% 的人之一。
</details>

### 跑馬燈

來自 1990 年代的網頁設計遺產，老派到極致就是潮！也是「這網站是我的地盤」的終極象徵。有哪個社群媒體會允許你在貼文中放跑馬燈的？

範例：

<Marquee>
🧻 本週特價！衛生紙買十送十！限時搶購中！ 🧻
</Marquee>
