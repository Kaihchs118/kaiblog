## 🎵 音樂製作與編曲 (Audio & Music)
這是我投入最多時間的領域，包含 DAW、打譜與虛擬音源。

- **Ableton Live 12 Suite** :
    嘗試看看新的專業的軟體...

- **Cubase 15 Pro** :
    主要的編曲軟體，教學資源豐富 !
    - **Groove Agent SE** :
    Cubase 內建的鼓機，還蠻好用的
    - **HALion Sonic** :
    Cubase 內建音源，雖然音色真實度普通，但耗電量低就是了
    - **Steinberg Download Manger** :
    Cubase 下載相關資源最好的地方
    - **Steinberg Library Manger** :
    管理 Steinberg 系列音色庫的地方

- **Finale 27** :
    用習慣了，雖然現在官方停止更新，但是我習慣了 ...
    - **Aria Player** :
    專為 Finale 設計的音樂播放器。
- **Musescore 4** :
    開源打譜軟體，但不太喜歡它把所有工具都堆在右邊的介面設計。
- **Audacity** :
    免費開源超好用的音訊編輯器，很適合製作 Podcast 
- **Logic Pro** :
    不常用，放著生灰🤣
- **UVR 5 (Ultimate Vocal Remover)** :
    去人聲超好用！免費開源，用本地模型
- **Vital** :
    介面現代且強大的合成器，非常好用。
    - **Helm** : 
        開源的合成器 ~
- **Kontakt 8** :
    軟體音源，最近像跳槽到硬體，歡迎大家推薦給我
    - **Kontakt Library Tools** :
        幫助我管理與入庫大量的 Kontakt 音源。

## 🎬 影像剪輯與視覺設計 (Visual & Video)
從影片剪輯到縮圖設計，這些是我的視覺工具箱。

- **Davinci Resolve** :
    找不到更好的剪輯軟體了！免費版限制極少、無浮水印，買斷制且永久更新。
- **Affinity Studio** :
    製作 YouTube 縮圖的主力，[可以看這篇文章介紹](/blog/affinity)。
- **OBS Studio** :
    超好用的開源螢幕錄影與直播軟體，功能極其強大。
- **GIMP 3** :
    [曾老師堅持使用的修圖軟體](/blog/731pc)，而且還堅持要用 2.6 古董版。
- **Canva** :
    製作簡報的工具。但我現在很不喜歡它，因為檔案存雲端常無故消失，且套版感太重。
- **Final Cut Pro** :
    基本上在Mac上用超省電，但我很不喜歡，因為是 `Mac` 獨佔

## ✍️ 筆記、生產力與文書 (Productivity)
我對筆記軟體的要求很高，目前的重心在「本地化」與「速度」。

- **[Anytype](https://anytype.io)** :
    **超推！** 開源且隱私度極高的筆記軟體。載入速度超快，這是我離開 Notion 的主因。[看介紹影片](https://youtube.com/playlist?list=PL8hRa_yxy1s357hLJfzMnzfMjSqTf3GNM)。
- **VSCodium** :
    目前所有的部落格文章都是在這裡撰寫完成的。跟微軟的不一樣，他是開源的 !
- **Raycast** :
    強大的啟動器，比系統內建的 Spotlight 好用太多，建議看 [PAPAYA 的教學](https://youtu.be/x6IcLAdUjSI?si=VECnUvUymeBcw5kC)。
- **Microsoft Word** :
    文書處理（未來會更新如何免費使用的方法）。
- **Microsoft Excel** :
    處理表格的工具。
- **Microsoft PowerPoint** :
    雖然有其他替代品，但 PPT 仍是職場標準。
- **Keynote** :
    蘋果內建的簡報軟體，美感很好，但通用性稍低。
- **Pages** :
    蘋果家的 Word，幾乎沒在用。
- **Numbers** :
    蘋果家的 Excel，同樣沒用過。

## 🤖 AI 應用與自研工具 (AI & Dev Tools)
這些是我與 AI 協作開發，或者是利用 AI 技術提升效率的工具。

- **[YTDownloader](https://kaiblog.itch.io/ytdownloader)** :
    **自研神器！** 我與 AI 合作開發的 YouTube 下載工具，真心推薦**不**好用。
- **Converter** :
    自己開發的檔案轉換工具，核心使用的是強大的 `FFmpeg`。
- **WhisperTRANS** :
    與 AI 開發的語音轉文字工具（需自行部署）。
- **Buzz** :
    使用 `Whisper` 模型，製作影片字幕時的轉錄好幫手。

## 🛠️ 系統增強與檔案管理 (Utilities)
讓 Mac 運作更順暢、傳輸更方便的小工具。

- **[YTDownloader](https://github.com/aandrew-me/ytDownloader)** :
     別人開發的跨平台 YouTube 下載工具，真心推薦**超**好用。(這別人推薦，但還是靠自己找到的)

- **[Pearcleaner](https://github.com/alienator88/Pearcleaner)** :
    **強烈推薦！** 每台 Mac 都該裝的清理軟體，解除安裝 App 非常好用。最近還更新可以搜尋檔案
- **[Blip](https://blip.net/)** :
    跨平台傳檔超方便，就算裝置在不同網路或遠距離也能傳。像我這個安卓仔...嗯 😐
- **[Keka](https://www.keka.io/zh-tw/)** :
    開源解壓縮工具，處理密碼壓縮檔或專業壓縮需求非常方便。
- **IINA** :
    Mac 萬能影音播放器，介面優雅，比系統內建好用一百倍。
- **[NDM](https://www.neatdownloadmanager.com/)** :
    下載加速器，目前終於可以在 Brave 上面強制下載，而且比FDM好用
- **RustDesk** :
    開源的遠端桌面軟體，穩定且好用。
- **Parallels Desktop** :
    在 Mac 上執行 Windows 虛擬機的首選，雖然要付費但性能極佳。
