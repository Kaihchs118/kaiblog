以下列出我所有 Mac 上面的軟體

1. Affinity Studio ㄧ 製作縮圖常用，免費，[可以看這篇文章](/blog/affinity)
2. Anytype ㄧ 超好用開源筆記軟體，可以看看[這部影片](https://youtube.com/playlist?list=PL8hRa_yxy1s357hLJfzMnzfMjSqTf3GNM)的說明，載入速度超快，這是為什麼我離開 Notion 的原因
3. Arena ㄧ 製作舞美 (VJ) 專用，目前還在初學階段
4. Aria Player ㄧ Finale 的音樂播放器
5. Audacity ㄧ 免費開源超好用的音訊編輯器，很適合製作 Podcast 
6. BiliBili ㄧ 想找一些中文教學影片就會去這裡找（YouTube 也會啦）
7. Blip ㄧ 開源跨平台傳輸檔案超方便，縱使**在不同網路**或者**距離很遠**也可以照樣傳送，缺點是所有裝置都需要安裝該軟體
8. Brave Browser ㄧ 開源且具有隱私的瀏覽器！連同步資料都是用類似加密貨幣一樣
9. Buzz ㄧ 可使用 `Whisper` 語音轉文字，製作影片的時候超方便
10. Canva ㄧ 製作簡報工具，後來很不喜歡，**所有東西都存在他的雲端，但我要下載時才發現檔案已經不見了（？🫡**，而且有的人都套用模板，不會學會**從零開始**製作簡報
11. Converter ㄧ 我自己跟AI合作開發的檔案轉換工具，主要是使用 `FFmpeg`
12. Cubase 15 Pro ㄧ 主要的編曲軟體，教學資源不錯！
13. Davinci Resolve ㄧ 找不到比他更好的剪輯軟體了！免費版限制超少，**不會有浮水印**，跨平台，終身買斷美金295+**永久提供更新**
14. Discord ㄧ 在某些時候聊天會用，視訊不卡頓，我覺得很不錯
15. Dorico ㄧ 想跳槽，但完全不會用！快捷鍵需熟練
16. Finale 27 ㄧ 習慣了，但是…停止提供更新了！
17. GIMP ㄧ [曾老師堅持使用的修圖軟體](/blog/731pc)，**而且還堅持用 2.6**
18. Github Destop ㄧ 在發布新文章時很好用，本網站就是使用 Github 架站的
19. Groove Agent SE ㄧ Cubase 內建的鼓機，還蠻好用的
20. HALion Sonic ㄧ Cubase 內建的虛擬音源，不真實，但耗電量低就是了
21. IINA ㄧ 開源萬能的播放器，比系統的好用100倍！
22. Keka ㄧ 開源好用的解壓縮工具，壓縮檔有密碼的，或者要製作專業壓縮檔案的，都很方便！
23. Keynote ㄧ 蘋果內建的簡報軟體，蠻好用的，但是
24. Kontakt 8 ㄧ 軟體音源，最近像跳槽到硬體，歡迎大家推薦給我
25. Kontakt Library Tools ㄧ 幫助我 Kontakt 入庫
26. LIne ㄧ 設計很差的通訊軟體，但大家都用他
27. Logic Pro ㄧ 不常用，放著生灰🤣
28. Microsoft Excel
29. Microsoft PowerPoint
30. Microsoft Word ㄧ 微軟文書軟體，比較常用 (未來有空更新免費使用方法)
31. Musescore 4 ㄧ 打譜軟體，但是我不喜歡他把所有東西都放在右邊！
32. NDM ㄧ 下載神器，到底這比較好還是 Motrix ?
33. Numbers ㄧ 沒用過，畢竟蘋果專用
34. OBS ㄧ超好用開源螢幕錄影軟體！
35. Obsidian ㄧ 現在輸入文章都是用它來完成！
36. Pages ㄧ 蘋果專用，一樣沒用過
37. Parallels Desktop ㄧ Windows 虛擬機，蠻好用的，但是要付費
38. Pearcleaner ㄧ **大推每台電腦都要裝**，清理超好用
39. Raycast ㄧ 啟動器，比 Spotlight 好用，教學可以看 PAPAYA 
40. Roblox ㄧ 我唯一玩的遊戲
41. Roblox Studio ㄧ 做遊戲，非常簡單，教學推薦：克拉克！
42. RustDesk ㄧ 開源遠端桌面，好用！
43. Steinberg Download Manger ㄧ Cubase 下載最遠的地方
44. Steinberg Library Manger ㄧ Cubase 管理資源庫的地方
45. UVR 5 一 去人聲超好用！免費開源，用本地模型
46. Vital ㄧ 好用合成器
47. VSCodium ㄧ 超好用文字編輯器，開源而且不會被微軟綁架
48. WhisperTRANS ㄧ 我與 AI 開發的語音轉文字工具，但要自己部署
49. [YTDownloader](https://kaiblog.itch.io/ytdownloader) ㄧ 我與 AI 開發的YouTube 影片下載神器，超級好用！！！！
